Employee Directory

This project is a simple React application that displays a list of employees in a card format. The employee data is fetched from an external API and displayed dynamically on the page.

Features
- Fetches employee data from an API
- Displays employees in reusable card components
- Search functionality to filter employees by name or email
- Responsive card layout

Technologies Used
- React
- JavaScript
- Vite
- CSS
- JSONPlaceholder API

How to Run the Project

1. Clone the repository
2. Navigate to the project folder
3. Install dependencies

npm install

4. Start the development server

npm run dev

5. Open the project in your browser using the local server link shown in the terminal.

API Used
The project uses a free fake REST API to fetch employee data.

https://jsonplaceholder.typicode.com/users
